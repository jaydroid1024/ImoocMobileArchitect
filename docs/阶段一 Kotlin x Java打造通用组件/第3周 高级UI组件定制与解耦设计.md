    - 提供通用的API，支持自定义Head

2. 疑难点分析

## HiBanner组件开发

1. 需求分析
    - 支持自动轮播
    - 支持轮播速度设置
    - 支持UI以及指示器高度定制
    - 不耦合图片加载库

2. 疑难点分析
    - 作为有限的item如何实现无限轮播    
    - 指示器的高度定制
    - 将网络图片库与Banner组件解耦
    - 如何设置ViewPager的滚动速度
